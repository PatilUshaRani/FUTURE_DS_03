# Marketing Funnel & Conversion Analysis

## Overview
This project analyzes marketing funnel data to understand how users move from Visitors to Leads to Customers. The goal is to find drop-offs, check channel performance, and improve conversion.

## Objectives
- Analyze funnel stages (Visitors → Leads → Customers)
- Calculate conversion rates
- Identify best performing channels
- Find where users drop off

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Line)

## Dataset
Columns used:
- Date
- Channel
- Visitors
- Leads
- Customers

## Analysis

### Funnel Analysis
Shows drop from Visitors to Leads to Customers.

### Channel Performance
Compares number of customers from each channel.

### Conversion Rate
Calculated using:
Customers / Visitors

### Trend Analysis
Tracks daily performance of Visitors, Leads, and Customers.

## Key Metrics
- Total Visitors: 35,370
- Total Leads: 6,305
- Total Customers: 1,444
- Visitor to Lead Conversion: 18%
- Lead to Customer Conversion: 23%

## Insights
- Organic channel gives highest customers
- Email has better conversion rate
- Major drop happens between Visitors and Leads
- Performance increases over time

## Conclusion
This analysis helps understand funnel performance and identify areas to improve conversion and customer acquisition.
